#Manage 

---

Find some shortcuts. 
Note: These shortcuts are only valid in the obsidian application.

| definition              | hotkey            |
| ----------------------- | ----------------- |
| search - file in vault  | alt + f           |
| search - text in vault  | alt + o           |
| change vault            | alt + v           |
| show graph              | alt + g           |
| create new note         | alt + t           |
| create header           | alt + 1/2/3/4/5/6 |
| create purple highlight | alt + p           |
