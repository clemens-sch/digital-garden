## # Pointer ?

Pointers are references to storages (RAM) of a variable.
- Reference-variable

```c
// Pointer definition
char* pp1;
char *pp2;

// assign address to variable
char p = 5;
char *pp = &p;

// access pointer
*pp = 6;
```
