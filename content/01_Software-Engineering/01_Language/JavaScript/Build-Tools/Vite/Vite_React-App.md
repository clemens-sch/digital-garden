## # Create React Application
__using Vite__

_Make sure to have `NodeJS` (npm) installed!_


Create Project-Folder:

```cmd
npm create vite@latest vite-app-name

React

JavaScript

cd vite-app-name

npm i
```


Run Project:

```terminal
npm i

npm run dev
```

---

Run for production:

- creates "dist" folder

```terminal
nmp run build
```

Run for preview:

```terminal
npm run preview
```
